## Commands

Do everystep for production at once

```
npm run production
```

Compiling flow typed js

```
grunt babel
```

Compile API doc

```
grunt jsdoc2md
```

Check flow types

```
npm run flow
```

Run Tests

```
npm test
```

Add table of contents to readmes

```
grunt markdown_toc
```

Linting

```
grunt eslint
```
